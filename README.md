## ✨ 功能特性

### 📅 1. 日期加减计算
快速计算指定天数前后的日期，支持正负天数计算

- **输入**：基准日期、加减天数
- **输出**：计算后的日期及星期
- **场景**：项目截止日期、倒计时、日程安排

### 🌙 2. 农历国历互查
公历与农历双向转换，显示详细信息

- **国历转农历**：显示干支纪年、生肖、月份日期、节气、星座、年月日时干支
- **农历转国历**：输入农历年月日，转换为对应公历日期
- **支持闰月**：可勾选闰月进行转换
- **场景**：查询传统节日、农历生日、择吉日

### 📊 3. 日期间隔计算
精确计算两个日期之间的时间差

- **输出**：相差天数、周数、小时数
- **场景**：计算工龄、项目周期、距离特定日期还有多久

### 🌍 4. 全球时区转换
跨时区时间换算，覆盖100+全球城市时区

- **覆盖地区**：
  - 中国及港澳台（北京、上海、香港、澳门、台北）
  - 亚洲其他地区（东京、首尔、新加坡、曼谷、迪拜等20+城市）
  - 欧洲地区（伦敦、巴黎、柏林、莫斯科等17+城市）
  - 美洲地区（纽约、洛杉矶、芝加哥、多伦多等24+城市）
  - 大洋洲地区（悉尼、奥克兰、斐济等10+城市）
  - 非洲地区（开罗、约翰内斯堡等5+城市）
- **场景**：国际会议、跨国协作、旅行计划

### ⏱️ 5. Unix 时间戳转换
时间戳与日期时间双向转换

- **时间戳 → 日期**：支持秒级和毫秒级
- **日期 → 时间戳**：支持 `YYYY-MM-DD HH:MM:SS` 格式，精确到秒
- **一键获取**：快速获取当前时间戳
- **场景**：开发调试、日志分析、数据迁移

### ⏰ 6. 最早/晚打卡时间
批量处理打卡记录，快速找出最早和最晚时间

- **输入格式**：每行一个，格式为 `YYYY-MM-DD HH:MM:SS`
- **输出**：最早/最晚的打卡记录及统计信息
- **场景**：考勤统计、工时计算、班次分析

## 🚀 快速开始

### 安装方法

1. 打开 [uTools](https://u.tools/)
2. 进入「插件中心」
3. 搜索「日期时间」或直接安装本插件
4. 或在 uTools 开发者模式中加载本插件目录

### 使用快捷指令

- `日期加减` / `date` - 日期加减计算
- `农历转换` / `lunar` - 农历国历互查
- `日期间隔` / `datediff` - 日期间隔计算
- `时区转换` / `timezone` - 全球时区转换
- `时间戳` / `timestamp` - Unix 时间戳转换
- `时间段` / `打卡时间` - 打卡时间处理

## 🛠️ 技术栈

- **前端框架**：纯 HTML/CSS/JavaScript（无框架依赖）
- **日历库**：[lunar-javascript](https://github.com/6tail/lunar-js) - 农历转换
- **时区处理**：原生 `Intl.DateTimeFormat` API
- **开发模式**：uTools 开发模式直连（无构建工具）

## 📝 使用示例
